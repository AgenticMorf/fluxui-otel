# Flux UI Otel

Flux UI OpenTelemetry observability dashboards for Laravel Livewire. Dashboard, traces, metrics, and services viewers backed by Jaeger, Loki, and Prometheus.

## Features

- Dashboard with traces, metrics, logs overview
- Traces viewer (Jaeger)
- Metrics viewer (Prometheus)
- Services viewer
- Otel ingest endpoint for browser telemetry
